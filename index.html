Asterisks Begone
When enabled, an "Asterisks Begone" button will be added to the character edit page. Clicking it will remove asterisks from all greetings and examples of dialogue.

When "Check for character actions" is enabled, the extension will detect if any greetings or examples contain character actions wrapped in asterisks (like *{{char}} nods*) mixed with regular text and preserve them by not removing any asterisks in that text.
It's smart enough to distinguish between legitimate character actions and excessive asterisk usage. When everything except dialogue is wrapped in asterisks, it will still clean them up.

Warning: Please back up your character card first before proceeding.
