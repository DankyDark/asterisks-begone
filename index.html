Asterisks Begone

Adds an "Asterisks Begone" button to the character edit page to remove asterisks from greetings and examples.

Preserves legitimate character actions (like *{{char}} nods*) mixed with regular text.

Smart detection can distinguish between character actions and excessive asterisk usage. When most paragraphs are wrapped in asterisks, it will clean them up.

Disable this setting to remove all asterisks regardless of content.

Warning: Please back up your character card before proceeding.
